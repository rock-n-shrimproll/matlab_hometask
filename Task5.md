#### Задание №2 ####
Решить систему линейных уравнений
```math
\begin{equation*}
 \begin{cases}
   7x+5y-2z = 5 \\
   y - 2z = 7 \\
   5x+3y+3z = -1
 \end{cases}
\end{equation*}
```
Решение:
```matlab
x = [7, 5, -2; 0, 1, -2; 5, 3, 3];
b = [5; 7; -1];
linsolve(x, b);
```

Ответ:
```matlab
ans =

   -4.0870
    6.6522
   -0.1739
```

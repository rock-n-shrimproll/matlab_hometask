#### Задание №2 ####

```math
\begin{equation*}
 \begin{cases}
   6x+y+4z = 6 \\
```

#### Задание №7 ####
Найти все корни и построить график:
```tex
{\frac 1 n}^x = \log_{\frac 1 n}x, n = 15.9
```

Решение:
```matlab
syms x
n = 15.9
f = (1/n)^x == log(x)/log(1/n)
ran = 0+0.6 .*rand(5,1)
ran = sort(ran)
for i=1:6
  vpasolve(f,x, [ran(i,1) ran(i+1,1)])
end
```
